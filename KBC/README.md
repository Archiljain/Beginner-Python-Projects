🎮 Kaun Banega Crorepati (KBC) - Python Game
A simple console-based quiz game inspired by the famous TV show "Kaun Banega Crorepati" (KBC). Play and test your general knowledge across multiple levels with increasing difficulty. Written in Python using conditional logic and interactive input/output.

🧠 Features
✅ Console based

💰 Increasing prize money with each correct answer

🛑 Game ends if the answer is wrong

🐍 Built in Python (ideal for beginners learning conditionals and loops)
