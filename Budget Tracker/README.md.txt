# 💰 Budget Tracker (CLI)

A simple **Command-Line Interface (CLI)** Budget Tracker app built with Python.  
Track your daily income, expenses, and right from your terminal!

## 📌 Features

- Add income and expenses
- View transaction history for one run only
- Check current balance
- Simple, beginner-friendly Python logic

## 🚀 Feel free to fork the repo and add more details to code and pull Request . 

### Requirements

- Python 3.x
- A terminal/command prompt

### How to Run

1. Clone the repository:
   git clone https://github.com/your-username/budget-tracker.git
   cd budget-tracker
2. Run the script :
   python budget_tracker.py

🛠️ Technologies Used
Python (basic file I/O, loops, conditionals)

CLI (Terminal based interface)
