# Power Calculator (Python Script)

This is a simple Python script that calculates the power of a number by taking user input for the base and exponent.

## 🧮 How It Works

- Asks the user to enter a base number
- Asks for the exponent
- Calculates the power using `base ** exponent`
- Displays the result

## 🛠️ Requirements

- Python 3.x

No external libraries are needed.
